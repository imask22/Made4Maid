# Made4Maid

It is a platform to connect relevant workers who are in need of jobs with the people who require their daily chores to be done.

It has distance matrix API with an accuracy of about 85%

It is a scalable website with lots of Future Scope

# Technologies/Languages used 

JAVA - SERVER - PAGES

HTML

CSS3

AJAX

BOOTSTRAP

MYSQL

# How To Setup

Initiate the project in Eclipse or any of your favourite IDE

Put Src folder as it is and change the servercon folder -> change it to your project Name

Put Web Pages in WebContent folder of your eclipse IDE

Import the Database

Input your mailID for the message verification over the network

Run the Project
